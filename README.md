# 🤖 Polymarket Copy-Trading Bot

Автоматический копи-трейдинг бот для платформы [Polymarket](https://polymarket.com).
Отслеживает сделки выбранных трейдеров и копирует их с настраиваемым риск-менеджментом.

---

## 📋 Возможности

- **Мониторинг** двух топ-трейдеров каждые 30 секунд
- **Фильтрация** сделок по цене, размеру и активности рынка
- **Дедупликация** — каждая сделка копируется ровно один раз
- **Стоп-лосс** — автоматическое закрытие убыточных позиций
- **Telegram уведомления** о каждом событии
- **Веб-дашборд** на порту 5000 с реальными данными
- **DRY-RUN режим** — тест без реальных денег

## ⚡ Быстрый старт

### Требования

- Python 3.10 или новее
- Polygon-совместимый кошелёк с USDC
- Аккаунт на Polymarket с API ключами

### 1. Установка зависимостей

**Windows:**
```bat
pip install -r requirements.txt
```

**macOS / Linux:**
```bash
make install
# или
pip3 install -r requirements.txt
```

### 2. Настройка окружения

```bash
# Скопируйте шаблон
cp .env.example .env

# Откройте и заполните реальными данными
nano .env        # Linux/macOS
notepad .env     # Windows
```

Обязательные поля в `.env`:

| Переменная | Описание | Где получить |
|---|---|---|
| `WALLET_PRIVATE_KEY` | Приватный ключ Polygon кошелька | MetaMask → Account details |
| `WALLET_ADDRESS` | Публичный адрес (0x...) | MetaMask |
| `CLOB_API_KEY` | Ключ CLOB API | polymarket.com → Profile → API Keys |
| `CLOB_API_SECRET` | Секрет CLOB API | Там же |
| `CLOB_API_PASSPHRASE` | Парольная фраза API | Там же |
| `TELEGRAM_BOT_TOKEN` | Токен Telegram бота | @BotFather → /newbot |
| `TELEGRAM_CHAT_ID` | ID вашего чата | @userinfobot |

### 3. Тест подключений (dry-run)

```bash
make test
# или
DRY_RUN=true python bot.py
```

Бот запустится, проверит все подключения и будет логировать обнаруженные сделки **без реального исполнения**. Убедитесь что:
- ✅ Polymarket Data API: доступен
- ✅ Telegram Bot: @ваш_бот
- ✅ Баланс USDC отображается корректно

### 4. Реальный запуск

```bash
# Установите DRY_RUN=false в .env файле, затем:
make run
# или
python bot.py
```

### 5. Веб-дашборд

После запуска бота откройте в браузере:
```
http://localhost:5000
```

Дашборд показывает статус, позиции, PnL и историю сделок в реальном времени.

## ⚙️ Параметры конфигурации

Все параметры задаются в `.env` файле или напрямую в `config.py`:

| Параметр | Дефолт | Описание |
|---|---|---|
| `DRY_RUN` | `true` | Режим без реальных сделок |
| `MAX_POSITION_USD` | `10.0` | Максимальный размер позиции ($) |
| `MIN_ENTRY_PRICE` | `0.05` | Минимальная цена входа (5¢) |
| `MAX_ENTRY_PRICE` | `0.70` | Максимальная цена входа (70¢) |
| `COPY_RATIO` | `0.5` | Доля от объёма оригинала (50%) |
| `MIN_COPY_SIZE_USD` | `1.0` | Минимальный размер для исполнения ($) |
| `MAX_OPEN_POSITIONS` | `20` | Лимит одновременных позиций |
| `POLL_INTERVAL_SEC` | `30` | Интервал опроса трейдеров (сек) |
| `STOP_LOSS_PERCENT` | `0.80` | Порог стоп-лосса (80% от входной цены) |
| `POLYGON_RPC` | polygon-rpc.com | RPC узел Polygon |
| `LOG_LEVEL` | `INFO` | Уровень логирования |

## 🛡️ Механизмы безопасности

**Фильтрация сделок:**
- Только BUY ордера
- Цена в диапазоне MIN_ENTRY_PRICE – MAX_ENTRY_PRICE
- Расчётный размер ≥ MIN_COPY_SIZE_USD
- Не превышен лимит MAX_OPEN_POSITIONS
- Рынок активен (проверка через Gamma API с кэшем на 5 мин)
- Дедупликация по ID транзакции

**Стоп-лосс:**
- Фоновый поток проверяет каждые 60 секунд
- Закрывает позицию если цена упала ≥ 20% от входной (настраивается)
- Уведомление в Telegram при срабатывании

## ⚠️ Важные предупреждения

1. **Всегда начинайте с DRY_RUN=true** и убедитесь что логика работает корректно
2. **Никогда не коммитьте `.env` файл** — он содержит приватный ключ
3. **Тестируйте с небольшими суммами** перед увеличением MAX_POSITION_USD
4. **Прошлые результаты трейдеров не гарантируют будущих**
5. **Polymarket — предсказательный рынок**, ваш капитал находится в зоне риска

## 🐛 Решение проблем

**`ImportError: py_clob_client не найден`**
```bash
pip install py-clob-client
```

**`web3.py не установлен — баланс не проверен`**
```bash
pip install web3
```

**Telegram не работает:**
- Проверьте TELEGRAM_BOT_TOKEN (запросите у @BotFather)
- Проверьте TELEGRAM_CHAT_ID (получите у @userinfobot)
- Убедитесь что вы написали боту хотя бы одно сообщение

**`CLOB API недоступен`:**
- Проверьте CLOB_API_KEY/SECRET/PASSPHRASE
- Проверьте что аккаунт верифицирован на Polymarket
- API ключи создаются в: polymarket.com → Profile → API Keys

**Бот не видит сделок трейдеров:**
- При первом запуске бот **игнорирует старые** сделки (только новые после запуска)
- Дождитесь активности трейдеров (может занять часы или дни)
- Проверьте логи: `make logs`
